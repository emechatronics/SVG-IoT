An HTML table, within a foreignObject provides an efficient means of automatically aligning HTML input elements, buttons, etc via the HTML table element, rather than attempting to position each element in the SVG drawing, which can be tedious.
NOTE: This works well in all browsers, except Firefox. It corrupts the parent HTML.
Scenerio:
This substantial and dynamically changing foreignObject includes tables, divs, and input elements. It represents a substation electrical monitoring panel. The input values are randomly created/updated every second.
